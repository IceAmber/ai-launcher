# AI Launcher v1.0.0

## 🎉 首次发布

AI Launcher 是一款基于人工智能的 Android 桌面启动器，集成了先进的语音识别和自然语言处理技术。

## ✨ 主要功能

### 🎤 语音控制
- 支持中文和英文语音识别
- 自然语言理解，像聊天一样控制手机
- 语音启动应用、发送消息、设置闹钟等
- 离线语音识别（需下载本地模型）

### 📲 智能应用管理
- 自动学习常用应用，显示在桌面顶部
- 最多显示 8 个常用应用（2 行 × 4 列）
- 快速搜索和启动任意应用

### 🎨 个性化定制
- **科技背景** - 默认科幻风格网格背景
- **图片壁纸** - 从相册选择图片作为壁纸
- **渐变背景** - 16 种预设渐变 + 自定义颜色
- **深色模式** - 自动跟随系统或手动切换

### ⚙️ 灵活配置
- 支持多种 AI 模型后端：
  - 云端模型（OpenAI、通义千问、DeepSeek 等）
  - 本地模型（MLC LLM，完全离线）
- 自定义 API 地址和密钥

## 🔒 隐私保护

- ✅ 所有数据保存在设备本地
- ✅ 不收集任何个人信息
- ✅ 语音识别在本地完成（使用本地模型时）
- ✅ 代码完全开源，可审计

## 📥 安装说明

1. 下载 `app-release.apk` 文件
2. 在 Android 设备上打开文件
3. 允许"安装未知应用"权限（如果提示）
4. 按照提示完成安装

## 📝 系统要求

- Android 5.0+ (API 21+)
- 建议 4GB+ 内存（使用本地模型时）

## 🐛 已知问题

- 首次启动可能需要加载时间
- 本地模型需要额外下载（约 1.5GB）
